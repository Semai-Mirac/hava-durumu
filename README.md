# Hava Durumu

Java Swing ile yazilmis hava durumu uygulamasi. OpenWeatherMap API kullanir.

## Kurulum

1. [OpenWeatherMap](https://openweathermap.org/api) adresinden ucretsiz API key alin.

2. Proje dizininde `config.properties` dosyasi olusturun:`n   `cp config.properties.example config.properties`

3. `config.properties` icine API key'inizi yazin:
   `pi.key=BURAYA_API_KEYINIZI_YAZIN`

   Alternatif olarak `OPENWEATHER_API_KEY` ortam degiskeni tanimlayabilirsiniz.
